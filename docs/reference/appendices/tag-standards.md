# Supported RFID Tag Types & Standards

<div className="badge-reference">REFERENCE</div>

**Audience:** Solution Builder

EPC Gen2 / ISO 18000-63. Memory banks: RESERVED, EPC, TID, USER. Tag access operations: READ, WRITE, ACCESS, LOCK, KILL (all supported via accessOperations).
