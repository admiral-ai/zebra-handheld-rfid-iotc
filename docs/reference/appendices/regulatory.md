#### Region codes

[DIAGRAM: D-20.5.A. world map shaded by region code]

| Region code | Description | Frequency band (MHz) |
|---|---|---|
| `US` | United States | 902–928 |
| `EU1` | European Union (ETSI) | 865.6–867.6 |
| `EU2` | European Union (extended) | 916.1–918.9 |
| `IN` | India | 865–867 |
| `JP` | Japan | 916.7–920.9 |
| `BR` | Brazil | 902–907.5, 915–928 |
| `CN` | China | 920.5–924.5 |
| `KR` | Republic of Korea | 917–920.8 |
| `AU` | Australia | 920–926 |

The full supported list and frequency tables are maintained at the Zebra regulatory portal. Country approvals may change; consult the portal for the current authoritative list before deploying to a new country.

#### Compliance certifications

The RFD40 and RFD90 carry FCC, ISED (Canada), CE (EU), MIC (Japan), and other regional certifications. Specific certification status per region is in the device's regulatory label and on the Zebra product page.
