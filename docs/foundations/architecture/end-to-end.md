# 2.1 About the End-to-End System

<div className="badge-explanation">EXPLANATION</div>

**Audience:** All

Five hops from tag to application: tag → sled → Bluetooth → host → Wi-Fi/cellular → broker → application. Each hop has its own failure modes.
